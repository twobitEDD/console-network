# Super Tic-Tac-Toe

The reference project for `@twobitedd/console-network`. It's a nested tic-tac-toe variant — you play 3×3 boards inside a 3×3 meta-board, and the cell you played on determines the sub-board your opponent must answer on.

Serves as a worked example of how a full game wires into the console: board view goes in the viewport, match history in α, roster in β, outcome card in MODAL, game clock in the HUD.
